# rules-us-vt

Vermont tax and benefit rules encoded in Cosilico DSL.

## Structure

```
statutes/     # Vermont Statutes Annotated (VSA)
regulations/  # Vermont Administrative Rules
```

## Sources

- [Vermont Statutes Online](https://legislature.vermont.gov/statutes/)
- [Vermont Secretary of State - Administrative Rules](https://sos.vermont.gov/administrative-rules/)
